My Bucket List

The Bucket List app is a full-stack web application I created with my group. The app allows users to create a list of activities they would like to do; search for activities using the Yelp API; add a search item to their bucket list, and view all bucket list's items they completed or did not complete. This app was created using JavaScript, jQuery, HTML/CSS, Bootstrap, Sass, AJAX, Node.js, Express, MongoDB, and Mongoose, and Handlebars.
